# File-Manager
Файловый менеджер на python с текстовым интерфейсом
Этот проект представляет собой текстовый файловый менеджер, разработанный на языке Python. Он позволяет пользователю взаимодействовать с файлами и папками внутри выделенной рабочей директории. Программа поддерживает создание, удаление, чтение, запись, копирование, перемещение и переименование файлов и папок.

Дополнительно реализована функция архивации и разархивации файлов и папок.

Основные возможности
Создание и удаление файлов и папок
Чтение и запись файлов
Переименование, копирование и перемещение файлов и папок
Просмотр содержимого рабочей директории и всех поддиректорий
Архивация файлов и папок в .zip
Разархивация .zip архивов
Все действия ограничены рабочей директорией, что предотвращает случайное изменение других файлов системы
